if语句和if else语句

if语句
if(判断语句)
表达式
如果判断语句为真就执行表达式

if else语句

if（判断语句）
表达式1
else
表达式2
例如：

、# include <stdio.h>
main()
{
 int age;
 printf(“请输入你的年龄：”);
 scanf(“%d”, &age);
 if(age>=18){
 printf(“恭喜，你已经成年\n”);
 }else{
 printf(“抱歉，你还未成年\n”);
 }
}
程序分析
这是一个判断是否成年的程序，如果if(age>=18)为真则执行 恭喜，你已经成年 如果为假则执行 抱歉，你还未成年
注意else与向上最近的为匹配的if匹配

逻辑运算符

&&  与
||  或
！  非
假设exp1和exp2是两个简单的关系表达式那么
exp1&&exp2 都为真是整判断式才为真其余都为假
exp1||exp2 都为假是整判断式才为真其余都为真
！exp1 如果exp1为真时则变为假 如果exp1为假时则变为真
条件运算符： ？：
语句1？A2：A3
如果语句1为真则取A2的值否则取A3的值
