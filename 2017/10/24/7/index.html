1.4 C语言基础概念2

发表于2017-10-24更新于2019-10-01

其它整数类型
Short int 型占用的储存空间可能比int少，常用于小范围数值，short int最少占16位
Long int 型占用的储存空间可能比int多常用于大数值的场合，long int最少占32位
Long long int 型占用的空间比long int更长，适用的范围也更大，long long int 最少占64位
Unsigned int型是无符号整型，也就是这个整型没有负数 允许的取值范围是0~65535

转义字符
顾名思义就是把编译器不能识别的特殊字符通过别的方式转义出来，例如：
\a 警报
\b 退格
\f 换页
\t 水平制表符
\r 回车
\n 换行
\v 垂直制表符
\\ 一个反斜杠
\ ’ 输出一个单引号
\ “ 输出一个双引号
\ ? 输出一个问号
\ 0oo 八进制（oo必须是有效的八进制）
\xhh 十六进制（hh必须是有效的八进制）

printf()
转换说明及其打印printf（）的输出结果
%C：单个字符
%D：有符号十进制整数
%F : 浮点型十进制计数法
%S：字符串
%X：无符号十六进制整数

scanf()
ANSIC码Scanf( ) 的转换说明
%C: 把输入解释成字符型
%D：把输入字符解释成十进制整数
%F: 把输入字符解释成浮点数
%f：把输入字符解释成浮点数
%O：把输入解释成有符号的八进制整数
%S：把输入解释成字符串
%X：把输入解释成有符号十六进制
%x：把输入解释成有符号十六进制
注意：（无符号：只有正没有负）
