vmware workstation（vm虚拟机）的使用

发表于2017-11-26更新于2019-10-01

vmware workstation 下载地址：http://rj.baidu.com/soft/detail/13808.html?ald

vm虚拟机简单来说就是在真实机上模拟多个计算机系统，操作与真机上的操作无异，但是它因为是在一台真机上模拟多个系统，所以它会占用大量的内存和cup，如果电脑的性能不是太强不建议用vm虚拟机
下面我来介绍vm虚拟机的使用方法

创建虚拟机有两种方式一是直接在主页上点击创建新的虚拟机二是工具栏点击文件点击新建虚拟机，
如果你已经创建完虚拟机可以在主页上点击打开虚拟机同样在工具栏文件点击打开效果是一样的

工具栏的第二个编辑 里面的剪切复制粘贴都是从真机向虚拟机内进行剪切复制粘贴
虚拟机网络编辑器 就是为了让虚拟机可以想真机一样联网而模拟出来的虚拟网卡，在虚拟网络编辑器中
有多种网络连接模式自己可以经行自定义
而首选项故名思意，就是设备的第一个选择对象，里面可以对硬件的优先级进行更改

工具栏第三个查看，为了调整用户在使用虚拟系统时的视觉感而出现的工具
这里我主要说一下unity 和 控制台视图
Unity 是VM虚拟机一种无缝显示方式 虚拟机系统 开启Unity模式后,就可以在真实机桌面上显示虚拟机系统的应用程序
控制台视图 就是在一个界面下显示多个虚拟机界面

工具栏第四个虚拟机 里面包含了电源 可移动设备 暂停 发送 快照 管理 设置
电源内的挂起计算机就在当前位置产生一个快照，等下次点继续运行客户机时，重新在快照当前保存的时刻恢复当前状态 关闭 重启按字面意思理解就可以了
在可移动设备内 可以进行可移动设备的插入弹出 比如u盘 移动硬盘等等都算是可移动设备
这个送功能就是向虚拟机内发送 ctrl+shift+del 防止与真机冲突
快照 简单点说它就相当于一个照片 下次可以从这个照片这个位置继续当前的操作
管理 里面有克隆 和 清除 克隆就是克隆出一个一模一样的虚拟机但克隆也分为两种 一个是链接克隆 就是在原来的虚拟机基础上克隆出来的虚拟机 如果原来的虚拟机被删除了 这个链接克隆的就不能用了 还有一个是完整克隆与之前的母本虚拟机完全独立 即使母本被删了 这个完整克隆出来的也还能用 完整克隆的缺点就是占用空间大 链接克隆占用空间小

工具栏第四个虚拟机 设置 我要单独来说明一下 如果你新建好了虚拟机 这个设置可以在两个地方打开看图片操作

点开设置 里面可以对硬件进行 一一编辑 可以编辑 内存 cpu 硬盘 等等。

在这里重点说一下 网络适配器 网络适配器可以更改虚拟机的网络连接方式 这里如果你不太会配置 点击NET模式 然后让虚拟机自动联网获取ip就可以了
NAT模式 利用虚拟的NAT设备以及虚拟DHCP服务器来使虚拟机连接外网
桥接模式就是将主机网卡与虚拟机虚拟的网卡利用虚拟网桥进行通信。在桥接的作用下，类似于把物理主机虚拟为一个交换机，所有桥接设置的虚拟机连接到这个交换机的一个接口上，物理主机也同样插在这个交换机当中，所以所有桥接下的网卡与网卡都是交换模式的，相互可以访问而不干扰。在桥接模式下，虚拟机ip地址需要与主机在同一个网段，如果需要联网，则网关与DNS需要与主机网卡一致。
仅主机模式 把虚拟机和真机连在同一局域网下 如果想要联网就需要真机给虚拟机共享网卡
