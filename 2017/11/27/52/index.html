Linux常用操作命令 (进阶篇) 1.3

发表于2017-11-27更新于2019-10-01

ls -a 列表显示当前目录下的文件和目录
ls -l 以长格式方式列表显示结果

mkdir -p
当你要创建的目录包含子目录时，你需要使用 -p 参数。假如mkdir找不到父目录，那么这个参数会首先帮助创建父目录，然后再创建子目录

mkdir -p 1/2/{3,4}/abc

touch abc bcd
创建两个0字节的问件 abc bcd

rm -f
有提示强制删除 可强制删除目录下的所有文件
rm -rf
无提示强制删除 可强制删除目录下的所有文件
rm -r
有提示删除但不是强制

ln a b 建立a的硬链接文件b
ln -s a b 建立a的符号链接文件b
ln -s 也可以指定目录进行创建链接

find /home -name “.txt”
在/home目录下查找以.txt结尾的文件名
find /home -name “my“
在/home目录下查找以my开头的文件名
