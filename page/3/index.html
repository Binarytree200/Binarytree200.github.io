2.2 C 控制语句：循环

发表于2017-10-26更新于2019-10-01

while循环

while(判断语句)
{
执行表达式
}
只有判断语句为真时才执行花括号内的表达式
例如
# include <stdio.h>
int main()
{
    int i = 1;
    while (i <= 100)
{
        printf (“%d\n”, i);
        i++;
   }
    return 0;
}
运行分析
while判断i是否小于等于100 如果满足执行表达式printf (“%d\n”, i);i++;输出i然后在给i加1一直输出到不满足i小于等于100 运行结果就是输出1到100的数字

其它整数类型
Short int 型占用的储存空间可能比int少，常用于小范围数值，short int最少占16位
Long int 型占用的储存空间可能比int多常用于大数值的场合，long int最少占32位
Long long int 型占用的空间比long int更长，适用的范围也更大，long long int 最少占64位
Unsigned int型是无符号整型，也就是这个整型没有负数 允许的取值范围是0~65535

常量和变量

其他数据类型在程序运行的过程中可能被赋值或改变的量成只为 变量
