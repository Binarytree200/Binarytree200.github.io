mysql 基本语法（学习sql注入入门）

发表于2020-09-22更新于2020-09-22

菜鸟MYSQL学习地址 https://www.runoob.com/mysql/mysql-tutorial.html
备注数据库：text_db
查看数据库（所有的） SHOW DATABASES;
创建数据库 CREATE DATABASE text_db;
查看指定数据库 SHOW CREATE DATABASE text_db;
使用数据库 USE text_db;
删除数据库 DROP DATABASE text_db;
备注:创建表 创建列的时候先用USE进入新建的数据库再进行创建
创建表 创建列 CREATE TABLE name_table(name_lie INT(11),name_lie2 INT(11),name_lie3 INT(11));

查看所有的表 SHOW TABLES;
增加表内容 插入数据 insert into name_table(name_lie,name_lie2) VALUES (123,456);
查看表中数据 SELECT *FROM name_table;(查询所有的列) SELECT name_lie FROM name_table;(查询name_table表中的name_lie列)
删除一个表 DROP TABLE name_table;
更改表名字 RENAME TABLE name_table TO name_table2;
增加一个列 ALTER TABLE name_table ADD name_lie3;
删除一个列 ALTER TABLE name_table DROP COLUMN name_lie3;
修改一个列 ALTER TABLE name_table CHANGE name_lie3 name_change_1 INT(11);
修改列中的一条记录 UPDATE name_table SET name_lie=321 WHERE name_lie=123;注意:WHERE后面的是要替换的列的原始值
删除列中的一条记录 DELETE FROM name_table WHERE name_lie=123;
